# Sentiment Analysis
The code has two parts:
- The first part is an example given by the teacher, which classifies if a Twitter post is positive or negative. Logistic regression and MLP are two separated approaches for the classifcation.
- The second part is implemented by myself, based on the example. It classifies if the emotion of a sentence is joy or sadness. The training dataset is `combined_emotion.csv`, which can be also found in this Kaggle link: https://www.kaggle.com/code/ivanpopravka/sentiment-emotion-classification-analysis/input. I also try using Twitter dataset for testing the emotion model and vice versa.
